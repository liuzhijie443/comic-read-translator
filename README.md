# 漫译助手

漫译助手是一个面向网页漫画阅读场景的 Tampermonkey 用户脚本，基于 [fast-image-translator.js](https://github.com/akatsukikyouko/fast-image-translator) 的图片翻译能力继续扩展，重点补上了对 [ComicRead](https://github.com/hymbz/ComicReadScript) 阅读模式的自动适配。脚本可调用本地或远程的 OpenAI 兼容多模态接口，识别图片中的外语文本并翻译为简体中文，适合漫画、图文资讯、社交媒体截图等图片内容的快速阅读。

在  [ComicRead](https://github.com/hymbz/ComicReadScript) 模式下自动跟踪图片加载状态、按并发队列逐页翻译、缓存结果，并在失败后按策略重试。

油猴直接安装 [漫译助手](https://greasyfork.org/zh-CN/scripts/583619-%E6%BC%AB%E8%AF%91%E5%8A%A9%E6%89%8B)

## 功能介绍

- **ComicRead 自动翻译适配**：识别 `ComicRead` 阅读容器和页面图片状态，在进入阅读模式后自动调度翻译。
- **普通网页图片翻译**：对站内普通图片保留悬浮按钮交互，可手动触发翻译。
- **图片替换模式**：将翻译结果直接绘制回原图，适合漫画气泡、对白框等场景。
- **文字翻译模式**：在不适合替换原图的网站上，以纯文字弹窗或结果面板展示译文。
- **并发翻译**：使用并发请求对漫画进行快速翻译。
- **多模型配置管理**：支持保存多个模型配置，分别设置名称、API 地址、模型 ID、密钥和额外参数，并快速切换。
- **翻译缓存**：按漫画页面地址缓存翻译结果，重复打开同一页面时可直接恢复，减少重复请求。
- **失败重试机制**：自动翻译失败后会记录失败状态并按上限重试，也支持手动重试失败图片。
- **自动翻译状态面板**：在 `ComicRead` 模式下显示启用状态、运行中任务数、排队数量、失败重试数和当前页信息。
- **智能文本布局**：使用归一化坐标、字号分级、相邻文本合并、自动换行和底板绘制，尽量保持原图阅读节奏。
- **JSON 修复与兼容处理**：对模型返回的 JSON 做容错解析，降低多模态模型输出格式不稳定带来的失败率。

## 适用场景

- 在线漫画站的连续阅读与批量翻译
- 使用 `ComicRead` 作为阅读层的网站
- 外语新闻截图、论坛配图、社交媒体图片内容翻译
- 已部署本地多模态模型，希望低成本长期使用图片翻译的场景

## 使用方式

1. 在 Tampermonkey 中安装 `ComicReadTranslator.js`。
2. 首次使用时打开脚本设置面板，填写可用的 OpenAI 兼容接口地址、模型 ID 和密钥。
3. 在普通网页图片上悬浮会出现“译”按钮，左键可直接翻译，右键可切换文字模式、站点策略和打开设置。
4. 在支持 `ComicRead` 的阅读页中，可通过右下角自动翻译按钮启用或关闭自动译。

## 配置项说明

- **API 接口**：OpenAI 兼容服务地址，例如本地部署的多模态服务。
- **模型 ID**：实际用于识图翻译的模型名称。
- **密钥**：调用接口所需的 API Key。
- **额外参数**：会直接并入请求体的 JSON 字符串，可用于传入服务商自定义参数。
- **自动译并发数**：`ComicRead` 自动翻译同时处理的图片数量。
- **自动译最大重试次数**：单张图片失败后允许自动再次尝试的次数。
- **最多缓存多少本漫画**：限制缓存页面集合数量，超出后自动清理较旧记录。
- **新页面默认开启自动译**：控制新进入的 `ComicRead` 页面是否默认启用自动翻译。

## 交互说明

- **图片悬浮按钮**：普通图片悬浮后出现按钮，支持手动翻译。
- **图片右键菜单**：提供文字翻译、站点级一律文字翻译、图标显示开关和设置入口。
- **浏览器脚本菜单**：可对最后右键选中的图片执行翻译，也可打开全局设置面板。
- **ComicRead 状态卡片**：支持显示进度、折叠、拖动、固定位置，以及手动重试失败任务。

## 主要增强

- 面向漫画阅读流程的自动翻译脚本，增加 `ComicRead` 模式识别、图片状态监听与自动任务调度。
- 增加按页面维度的翻译缓存与恢复逻辑。
- 增加失败状态跟踪、自动重试与手动补救入口。
- 增加多模型配置管理，而不只是单一模型参数输入。
- 增加更完整的自动翻译状态可视化面板。
- 翻译后文字和气泡的显示效果。
